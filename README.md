# GitHub-User-Finder
## Описание
Программа `GitHub User Finder` - это простое desktop-приложение на Python, разработанное с использованием библиотеки Tkinter. Оно позволяет пользователям искать аккаунты разработчиков на GitHub, вводить никнейм в поле поиска и просматривать результаты. Найденные пользователи отображаются в виде таблицы с указанием их логина и URL аватара.
Ключевая особенность - возможность добавлять найденных пользователей в "Избранное". Список избранных сохраняется в файл `favorites.json`, что обеспечивает персистентность данных между запусками приложения. Также присутствует функционал для просмотра списка избранных пользователей.

## Основные возможности:
*   Поиск пользователей GitHub по никнейму.
*   Отображение результатов поиска в виде таблицы.
*   Добавление пользователей в список "Избранное".
*   Сохранение избранных пользователей в `favorites.json`.
*   Просмотр списка избранных пользователей.

  # Как использовать:
1.  **Запуск:**
    *   Убедитесь, что у вас установлен Python.
    *   Сохраните код в файле с расширением `.py` (например, `github_finder.py`).
    *   Запустите файл из командной строки: `python github_finder.py`
2.  **Поиск:**
    *   Введите никнейм пользователя GitHub в текстовое поле.
    *   Нажмите кнопку "Найти". Результаты появятся в таблице ниже.
3.  **Добавление в избранное:**
    *   Выберите пользователя из таблицы результатов поиска.
    *   Нажмите кнопку "Добавить в избранное" или дважды кликните по выбранному пользователю.
4.  **Просмотр избранных:**
    *   Нажмите кнопку "Показать избранных" для открытия отдельного окна со списком добавленных пользователей.

## Зависимости:
*   `tkinter` (обычно входит в стандартную библиотеку Python)
*   `requests` (необходимо установить: `pip install requests`)

## API GitHub:
Приложение использует публичный API GitHub для поиска пользователей. При каждом поиске выполняется GET-запрос к `https://api.github.com/search/users` с параметром `q`, содержащим введенный запрос.
